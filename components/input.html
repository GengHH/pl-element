# 金额输入

后端一般传入前端的值为分 前端需做一些金钱转换传给后端分为单位

元

传入后台金额：1231分

实际金额：12.31元

<template>
  <div>
    <pl-input cent v-model="amt" @change="change">
      <i name="suffix">元</i>
    </pl-input>
    <p>传入后台金额：{{amt}}分</p>
    <p>实际金额：{{realMoney}}元</p>
  </div>
</template>

<script>
export default {
  name: 'input-demo-03',
  data () {
    return {
      amt: '1231',
      realMoney: '12.31'
    }
  },
  methods: {
    change (val) {
      this.realMoney = val
    }
  }
}
</script>

# textarea

与el-input渲染textarea一致,添加type属性即可

<template>
  <pl-input v-model="textarea" type="textarea" :autosize="{ minRows: 2, maxRows: 4}"></pl-input>
</template>

<script>
export default {
  name: 'input-demo-04',
  data () {
    return {
      textarea: ''
    }
  }
}
</script>
